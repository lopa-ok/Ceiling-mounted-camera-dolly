# Ceiling Camera Dolly

## what is this
this is a motorized ceiling camera dolly i built from scratch using 2040 aluminum extrusion, a nema 23 stepper motor, and two esp32s talking to each other over lan. it carries a 4kg camera across a 3m rail mounted to the ceiling and is controlled with a physical handheld box

## why i made this
it started out by me wanting to make a camera dolly for some reason, but then i realized i wouldnt be using this thing much so asked my local church if they would need something like this, and they said yes, but they would want something thats ceiling mounted, so modified my design to be ceiling mounted!
